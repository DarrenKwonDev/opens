# CV 이력서 호스팅

이 저장소는 GitHub Pages를 통해 CV(이력서) PDF 파일들을 웹으로 호스팅합니다.

## 파일 목록

- `suhunkwon_cv_eng.pdf` - 영문 이력서
- `권수훈_cv_kor.pdf` - 한글 이력서

## 접근 방법

배포 후 다음 URL로 PDF 파일에 접근할 수 있습니다:
- `https://[username].github.io/suhunkwon_cv_eng.pdf`
- `https://[username].github.io/권수훈_cv_kor.pdf`

## 업데이트 방법

PDF 파일을 업데이트한 후:
```bash
git add .
git commit -m "Update CV"
git push
```
